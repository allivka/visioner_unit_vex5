# vislib

robotics c++ template library

## version: 0.0.23

## Description

A "vislib" library provides simple templated solutions for containers, simple memory management and utility and math functions.


## Installation

Vislib can be either manually added to project or installed with platformio package manager by adding "alivka/vislib" to lib_deps field of platformio.ini file, e.g.:

```ini
[env]
platform = ...
board = ...
framework = ...
lib_deps =
    alivka/vislib
    ...
```
